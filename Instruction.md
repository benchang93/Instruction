# Instruction of GitHub

## Command order

1.	cd Desktop
2.	mkdir filefoldername
3.	cd filefoldername
4.	git init
5.	git status
6.	git remote add origin https://github.com/username/myproject.git 
7.	git remote -v (to check if we can access online repository)
8.	git pull https://github.com/username/myproject.git  
9.	ls (show the file in your file fold)
10.	git add Instruction.docx
11.	git commit  -m “Instruction of GitHub”
12.	git push https://github.com/username/myproject.git
